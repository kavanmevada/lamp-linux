LAMP is a free, local server environment that can be installed under linux environment with just a few commands. LAMP provides them with all the tools they need to develop php website on their desktop PC for testing or development purposes.
Supported On:

Install :

wget -Nnv -q https://raw.githubusercontent.com/kavanmevada/lamp-linux/master/LAMP && sudo bash LAMP -install; rm -f LAMP

Remove :

wget -Nnv -q https://raw.githubusercontent.com/kavanmevada/lamp-linux/master/LAMP && sudo bash LAMP -remove; rm -f LAMP
